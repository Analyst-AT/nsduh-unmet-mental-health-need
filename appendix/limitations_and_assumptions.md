# Limitations and Assumptions

- Cross-sectional design: associations only; no causal inference.
- Self-report: possible recall/reporting bias.
- Public-use constraints: some contextual variables may be unavailable.
- Model specification: covariates included are standard demographic controls; additional confounders (e.g., income, geography, symptom severity) may influence estimates if available.

Assumption: Survey-weighted methods correctly account for NSDUH design using the appropriate weight, strata, and PSU variables from the NSDUH documentation.

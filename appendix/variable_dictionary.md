# Variable Dictionary (as used in this project)

Outcome
- **UNMET_MH**: Unmet mental health treatment need (0 = No, 1 = Yes)

Exposure
- **INSURED**: Insurance status (0 = Uninsured, 1 = Insured)

Covariates
- **IRSEX**: Respondent sex
- **CATAG3**: Age category (18–25, 26–34, 35–49, 50+)
- **NEWRACE2**: Race/ethnicity category
- **EDUHIGHCAT**: Education category

Notes
- Exact category labels and design variables (weight/strata/PSU) should be verified in the NSDUH 2024 public-use documentation.

# Data Access (NSDUH 2024)

This repository does **not** include NSDUH microdata.

To reproduce the analysis:
1. Obtain the **NSDUH 2024 public-use file** and documentation from SAMHSA.
2. Open/import the dataset in SPSS.
3. Use the syntax in `/code/` and create/load a Complex Samples Plan file (.csaplan) based on NSDUH design variables.
